2  * File: init.h

4  * This file contains code to check whether the Stanford C++ library has been
5  * initialized, and if not, to initialize it.
6  * This file must be included by every student-facing header in the Stanford
7  * C++ library to make sure that the library is initialized before it is used.
8  *
9  * There used to be a group of files such as private/main.h, main.cpp, and
10  * startup.cpp that contained various initialization code. These were removed
11  * and replaced with this style of initialization for the following reasons:
12  *
13  * - simplicity/consolidation
14  * - allow student to NOT include console.h and use plain text console
15  *
16  * @version 2018/08/28
17  * - refactor to use stanfordcpplib namespace and init.cpp
18  * @version 2018/07/03
19  * - add code to handle Qt GUI library initialization
20  * @version 2017/04/25
21  * - wrap library initializer in an #ifndef to avoid multiple declaration
22  *
23  * TODO: figure out how to support both 0-arg and 2-arg main()
24  */
