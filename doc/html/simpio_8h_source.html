31  * Function: getChar
32  * Usage: char c = getChar(prompt);
33  * --------------------------------
34  * Reads a complete line from <code>cin</code>, expecting the user to type a
35  * single character. If this is the case, that character value is returned.
36  * If the user types anything other than a one-character line of input,
37  * the user is given a chance to reenter the value.
38  *
39  * If supplied, the optional <code>prompt</code> string is printed before
40  * reading the value.
41  *
42  * The also optional <code>reprompt</code> argument provides an output message
43  * displayed each time if the user types a file that is not found.
44  * If no value is passed, defaults to, "You must type a single character. Try again.".
45  */
46 char getChar(const string& prompt = "",
47  const string& reprompt = "");

188 /*
189  * Reads a complete line from <code>cin</code> and treats it as a
190  * yes-or-no answer to a question. Returns <code>true</code> if the line
191  * typed begins with a 'y' or 'Y', and returns <code>false</code> if it begins
192  * with a 'n' or 'N'. Otherwise the user is given a chance to reenter the
193  * value.
194  *
195  * If supplied, the optional <code>prompt</code> string is printed before
196  * reading the value.
197  *
198  * The also optional <code>reprompt</code> argument provides an output message
199  * displayed each time if the user types a file that is not found.
200  * If no value is passed, defaults to,
201  * "Please type a word that starts with 'Y' or 'N'.".
202  *
203  * The also also optional <code>defaultValue</code> argument indicates what
204  * should happen if the user just presses Enter rather than typing Y or N.
205  * By default, the user is re-prompted, but if a defaultValue is passed here,
206  * pressing Enter will be equivalent to having typed that value.
207  * This is useful where the default Y/N answer is Yes or No and you want to
208  * let the user avoid typing.
209  */
210 bool getYesOrNo(const string& prompt = "",
211  const string& reprompt = "",
212  const string& defaultValue = "");
