35 /*
36  * Class: TokenScanner
37  * -------------------
38  * This class divides a string into individual tokens. The typical
39  * use of the <code>TokenScanner</code> class is illustrated by the
40  * following pattern, which reads the tokens in the string variable
41  * <code>input</code>:
42  *
43  *<pre>
44  * TokenScanner scanner(input);
45  * while (scanner.hasMoreTokens()) {
46  * string token = scanner.nextToken();
47  * ... process the token ...
48  * }
49  *</pre>
50  *
51  * The <code>TokenScanner</code> class exports several additional methods
52  * that give clients more control over its behavior. Those methods are
53  * described individually in the documentation.
54  */
