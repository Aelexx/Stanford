1 /*
2  * File: urlstream.h
3  * -----------------
4  * This file exports the <code>iurlstream</code> class, which
5  * implements an input stream for reading data from URLs.
6  * It is implemented as a thin wrapper around a file input stream (ifstream).
7  * The implementation simply downloads the URL's data using Java's java.net.URL class
8  * and saves it into a temporary file, then reads from a stream on that file.
9  * Upon close()ing the iurlstream, the temporary file is deleted.
10  * This is not an ideal implementation, but it is platform independent and easier to get working.
11  *
12  * @author Marty Stepp
13  * @version 2018/06/20
14  * - support for setting headers such as user agent
15  * - https URL support
16  * - changed string to const string&
17  * @version 2014/10/08
18  * - initial version
19  * @since 2014/10/08
20  */
